# Logo Assets

Place PNG logo files in this folder. **Any shape/size works** - the CSS centers and fits images automatically.

## Required Files

### Global Providers

- `google-maps.png` - Google Maps
- `google-search.png` - Google Search
- `bing-maps.png` - Bing Maps
- `apple-maps.png` - Apple Maps
- `openstreetmap.png` - OpenStreetMap
- `mapquest.png` - MapQuest
- `here.png` - HERE WeGo

### Regional Providers

- `baidu.png` - Baidu Maps (China)
- `amap.png` - Amap / Gaode (China)
- `yandex.png` - Yandex Maps (Russia/CIS)
- `naver.png` - Naver Maps (South Korea)
- `kakao.png` - KakaoMap (South Korea)
- `mapy-cz.png` - Mapy.cz (Czech Republic/EU)

### API Providers

- `usps.png` - USPS

## Notes

- Any image shape works (square, rectangle, etc.)
- CSS centers the image and fits it within the container
- Transparent backgrounds recommended
